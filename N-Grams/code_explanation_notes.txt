### Lines 27-36: Text preprocessing pipeline
```python
import re
corpus = []
for i in range(len(sentences)):
    review = re.sub('[^a-zA-Z]', ' ', sentences[i])
    review = review.lower()
    review = review.split()
    review = [lemmatizer.lemmatize(word) for word in review if not word in set(stopwords.words('english'))]
    review = ' '.join(review)
    corpus.append(review)
print(corpus)
```

**Line-by-line explanation:**

**Line 27: `import re`**
- Imports regular expressions library for pattern matching and text cleaning

**Line 28: `corpus = []`**
- Creates empty list to store processed sentences (cleaned corpus)

**Line 29: `for i in range(len(sentences)):`**
- Starts loop to process each sentence individually
- `len(sentences)` = 5, so loop runs 5 times (i = 0,1,2,3,4)

**Line 30: `review = re.sub('[^a-zA-Z]', ' ', sentences[i])`**
- Uses regex to remove all non-alphabetic characters
- `[^a-zA-Z]` matches anything that's NOT a letter (numbers, punctuation, brackets)
- Replaces them with space ' '
- Example: "PM Narendra Modi..." → "PM Narendra Modi "

**Line 31: `review = review.lower()`**
- Converts entire sentence to lowercase
- Example: "PM Narendra Modi" → "pm narendra modi"

**Line 32: `review = review.split()`**
- Splits sentence into list of individual words
- Example: "pm narendra modi" → ['pm', 'narendra', 'modi']

**Line 33: `review = [lemmatizer.lemmatize(word) for word in review if not word in set(stopwords.words('english'))]`**
- **List comprehension** that processes each word:
- `stopwords.words('english')`: Gets list of English stopwords
- `set(stopwords.words('english'))`: Converts to set for faster lookup
- `if not word in set(...)`: Keeps word only if it's NOT a stopword
- `lemmatizer.lemmatize(word)`: Applies lemmatization to each remaining word
- Example: ['pm', 'narendra', 'modi', 'is'] → ['pm', 'narendra', 'modi'] (removes 'is')

**Line 34: `review = ' '.join(review)`**
- Joins words back into single string with spaces
- Example: ['pm', 'narendra', 'modi'] → "pm narendra modi"

**Line 35: `corpus.append(review)`**
- Adds cleaned sentence to corpus list
- Builds final processed corpus

**Line 36: `print(corpus)`**
- Displays final cleaned corpus showing all processed sentences

## STEMMING SECTION (COMPLETED)

### Lines 38-52: Complete stemming implementation
```python
## steming the sentences
stemmer = PorterStemmer()
stemmed_corpus = []
for i in corpus:
    stemmed_words = []
    for word in i.split():
        stemmed_word = stemmer.stem(word)
        stemmed_words.append(stemmed_word)
    stemmed_sentence = ' '.join(stemmed_words)
    stemmed_corpus.append(stemmed_sentence)

print("Original corpus:")
print(corpus)
print("\nStemmed corpus:")
print(stemmed_corpus)
```

**Line-by-line explanation:**

**Line 38: `## steming the sentences`**
- Comment indicating beginning of stemming process

**Line 39: `stemmer = PorterStemmer()`**
- Creates new PorterStemmer object for stemming operations
- Note: We already created one earlier, but creating again for clarity

**Line 40: `stemmed_corpus = []`**
- Creates empty list to store stemmed sentences
- Will contain the same sentences but with words reduced to their stems

**Line 41: `for i in corpus:`**
- Loops through each cleaned sentence from the original corpus
- Processes each sentence individually

**Line 42: `stemmed_words = []`**
- Creates temporary list to store stemmed words for current sentence
- Resets for each new sentence

**Line 43: `for word in i.split():`**
- Splits current sentence into individual words
- Loops through each word in the sentence

**Line 44: `stemmed_word = stemmer.stem(word)`**
- Applies PorterStemmer algorithm to reduce word to its stem/root
- Example: "language" → "languag", "directed" → "direct"

**Line 45: `stemmed_words.append(stemmed_word)`**
- Adds stemmed word to temporary list
- Builds list of stemmed words for current sentence

**Line 46: `stemmed_sentence = ' '.join(stemmed_words)`**
- Joins stemmed words back into sentence string
- Reconstructs sentence with spaces between words

**Line 47: `stemmed_corpus.append(stemmed_sentence)`**
- Adds stemmed sentence to final stemmed corpus list
- Builds complete stemmed corpus

**Lines 49-52: Output comparison**
